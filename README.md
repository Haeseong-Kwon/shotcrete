# Shotcrete

Shotcrete는 PC와 모바일 환경을 모두 지원하는 고성능 반응형 서비스입니다.

## 프로젝트 개요
- **목표**: 사용자에게 최상의 성능과 직관적인 UX를 제공하는 웹 서비스 구축
- **핵심 가치**: 성능(Performance), 반응성(Responsiveness), 프리미엄 디자인(Premium Design)

## 기술 스택 (제안됨)
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **UI Components**: shadcn/ui
- **Animations**: Framer Motion

## 시작하기
```bash
npm install
npm run dev
```
